AAP 04 - Tabelas

Com base no projeto anterior, crie uma nova página HTML com uma tabela com os cursos da Fatec Tatuí e o período dos cursos oferecidos (Você pode consultar o site da Fatec para visualizar essa informação)

Feito a tabela, crie um hyperlink para navegar entre a página da tabela e a página da Arpanet. Você pode usar o comando "a href" e nomear o arquivo como "fatec.htlm" e o outro de "arpanet.html" - deve ser possível navegar de um para o outro com LINK.
