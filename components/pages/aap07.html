AAP 07 - Lista Suspensa

Dando andamento nas tarefas anteriores, em que foi proposto criar uma página com uma disciplina do curso de GTI em específico e na última AAP, implementar um CSS. Agora vamos automatizar, a ideia desta AAP é criar uma pequena autmoação no seu projeto. Para isso vamos utilizar o JavaScript.

Vamos usar a classe .hidden que será usada para controlar a visibilidade dos elementos na página, aplicando o estilo display: none; para ocultá-los. O JavaScript seleciona todos os botões com a classe .toggle-button e adiciona um evento de clique a cada um. Quando o botão é clicado, o script obtém o ID do elemento correspondente e verifica se ele possui a classe .hidden. Se tiver, a classe é removida, tornando o elemento visível; se não tiver, a classe é adicionada, ocultando o elemento. Isso permite que o usuário mostre ou oculte as seções "Ementa" e "Bibliografia" de forma interativa.
